# 算符优先分析法  
## 要求  
1.	任意输入一个文法G
2.	对文法中的每个非终结符自动生成并打印输出其FIRSTVT与LASTVT集合
3.	判断该文法是否为算符优先文法
4.	模拟分析过程  
	如输入一个句子，如果该句子合法，能够输出分析过程中每一步符号栈的变化情况以及根据当前最左素短语进行规约的过程，如果该句子非法，则进行适当的报错处理
